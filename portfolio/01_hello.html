Project Hello Quad

3D Graphics

Project Overview

This project is the first graphics demo for CS250, aiming to set up a foundational OpenGL rendering pipeline. The goal was to render an animated, texture-mapped quad on the screen, establishing the architecture for future 3D applications.

Tasks Completed

  • Implemented foundational OpenGL wrapper classes including IndexBuffer, VertexBuffer, VertexArray, and Texture.
  • Integrated time-based animation logic within the vertex shader to oscillate vertex positions, creating a wavy motion effect.
  • Configured fragment shaders to dynamically interpolate colors between vertex colors and a custom-drawn texture based on time.
  • Adapted the project to successfully build and run on both Windows x64 and Web targets (via Emscripten).

Reflection

This assignment provided a solid introduction to OpenGL rendering mechanics and state management. Working with WebGL compilation via Emscripten highlighted the necessity of writing robust, cross-platform shader code. One of the main challenges was correctly wrapping OpenGL's C-style API into a more modern, object-oriented C++ design, especially with correctly configuring Vertex Array object states. Successfully seeing the texture reload feature and shaders in action was incredibly rewarding.
