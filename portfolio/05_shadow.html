Project Shadow Mapping

3D Graphics

Project Overview

In this assignment, I implemented and integrated Shadow Mapping, a real-time graphics technique for rendering shadows cast by objects. A perspective light source is modeled as a camera, whose viewing frustum is used to render a depth map of the scene. During the main screen-space rendering pass, the distance from each fragment to the light is compared with the recorded depth value to determine whether the fragment is in shadow.

Tasks Completed

  • Implemented a robust depth texture generation routine (LoadAsDepthTexture) inside the Texture class, configuring clamping and comparison mode settings to eliminate edge artifacts.
  • Constructed precise camera coordinate transformation matrices (ViewMatrix and ToWorldMatrix) inside the Camera class using column-major vectors and dot products, avoiding generic matrix inversion calls.
  • Built and validated custom FrameBuffer targets to handle multi-pass rendering, dynamically linking depth and color texture attachments.
  • Implemented rendering control pipelines in C++: rendering scene geometry to the light depth buffer using polygon offset fill, rendering shadows with projective texture mapping via textureProj in shaders, and visualizing the light's camera frustum and depth map.
  • Authored vertex and fragment GLSL shaders (shadows.vert and shadows.frag) supporting perspective depth space projection, behind-light shadow elimination, exponential fog attenuation, and gamma-corrected colors.

Reflection

One of the primary challenges in shadow mapping is shadow acne, caused by numerical precision limits in the depth texture. I resolved this by applying a depth offset via GL_POLYGON_OFFSET_FILL during the depth pass to offset the fragment depths backward. Another challenge was reverse projection shadows where objects behind the light source cast spurious shadows. I implemented a boundary check in the fragment shader to test whether the shadow space coordinates fall behind the projection frustum (where homogeneous coordinate w < 0.0 or depth z < 0.0), successfully bypassing reverse shadowing. This homework deepened my understanding of spaces, projection matrices, and multi-pass rendering pipelines.
